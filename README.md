<!-- INSTALLATION -->
## Installation

1. Clone or Download the repository

	```
	$ git clone https://github.com/bizarrecode/angular-crud-rest-api.git
	$ cd angular-crud-rest-api
	```
2. Install Dependencies

	```
	$ npm install
  	```
