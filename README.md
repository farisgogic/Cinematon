## About The Project

<!-- INSTALLATION -->
## Installation

1. Clone or Download the repository

	```
	$ git clone https://github.com/farisgogic/Cinematon.git
	$ cd Cinematon
	```
2. Install Dependencies

	```
	$ npm install
  	```
